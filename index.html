Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para Negócio com o Streamlit

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimetno web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace global de restaurantes.

O resultado final do projeto foi um painel hospedado em um ambiente Cloud e disponibilizado através de um link web. O painel pode ser acessado por qualquer dispositivo conectado na internet.

As ferramentas utilizadas foram:

  • Python.
  • Jupyter Lab.
  • Termina.
  • Streamlit.
